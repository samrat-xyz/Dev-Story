# Next.js Fundamentals Practice Project

This project was built to practice and understand the core concepts of Next.js using the App Router.

##  Project Goal
The main goal of this project is to strengthen my understanding of Next.js fundamentals, including routing, layouts, and page-level features.

##  Concepts Practiced
- App Router structure
- Layouts and nested layouts
- File-based routing
- Dynamic routing
- Dynamic segment routing
- `loading.js` for route-level loading states
- `not-found.js` for custom 404 handling
